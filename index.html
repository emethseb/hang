Beállítások

Váltás azonnal érvényes. Az aktív gomb kék kerettel jelölt.
Szükséges minden hanghoz. Létrehozás: elevenlabs.io → bal felső menü → ElevenAPI → API Keys
Hogyan hozd létre saját hangod?
  1. Menj: elevenlabs.io → regisztrálj
  2. Kattints a Voices+ menüre, majd Create voice
  3. Válaszd az Instant Voice Clone (előfizetés) vagy Voice Design (ingyenes) opciót
  4. API kulcs: bal felső menü → ElevenAPI → API Keys → új létrehozása
  5. Hang ID: Voices → My Voices alatt, a hangod neve mellett: Copy Voice ID
Az AI funkciókhoz (mondat generálás, kézírás felismerés, párbeszéd) Groq API kulcs szükséges. Az előre beállított kulcs automatikusan be van töltve – csak akkor módosítsd, ha új kulcsot szeretnél használni.
Hogyan szerezd meg a Groq API kulcsot?
  1. Menj: console.groq.com
  2. Regisztrálj Google fiókoddal (ingyenes)
  3. API Keys menü → Create API Key
  4. Másold ki a kulcsot (gsk_… formátum)

Gyors mondat szerkesztése

Hangom – Súgó

Az alkalmazás hangjukat elvesztett személyeknek nyújt gyors, AI-alapú kommunikációs segítséget.

Gépelés & Azonnali felolvasás

A főoldalon szabadon begépelhetsz bármilyen szöveget. A Felolvasás gomb megnyomásával a program azonnal hangosan felolvassa. A mentés gombbal egyből a mentett mondatok közé kerül.

Gyors mondatok

Az oldal alsó részén előre beírt gyors mondatok találhatók kártyák formájában – egyetlen érintéssel felolvastatható. A Szerkesztés gombbal bármely kártyát testre szabhatod (köszönés, kérések, válaszok, stb.).

Mentett mondatok

Ide gyűjtöd az összes fontos, előre megírt mondatot: bemutatkozás, orvosi tünetek, ügyintézési információk. Szöveg alapján kereshetsz közöttük, szerkesztheted vagy törölheted őket, és azonnal felolvastathatsz bármelyiket.

AI mondatgenerálás

Írj be 2–4 kulcsszót (pl. „orvos, fájdalom, segítség"), és az AI azonnal teljes, értelmes magyar mondatokat generál belőlük – ragozás és nyelvtan nélkül. A generált mondatokat felolvastathatod vagy elmentheted. Groq API kulcs és internet szükséges.

Kézírásos bevitel

Ha a gépelés nehezedre esik, ujjal vagy ceruzával írhatsz a vászonra. A program AI segítségével felismeri a kézírást és szöveggé alakítja. A felismert szöveget felolvastathatod, a gépelő mezőbe küldheted, vagy elmentheted.

Intelligens párbeszéd

Indítsd el a mikrofont és tedd le a telefont magad mellé. A program érzékeli a kérdező hangját (orvos, hozzátartozó), felismeri a kérdést, majd az AI automatikusan generál 3 igenleges és 3 nemleges választ. Kiválasztod a legmegfelelőbbet és egy gombnyomással felolvastatod – gépelés nélkül.
